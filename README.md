El objetivo de este trabajo práctico es poder demostrar los conocimientos adquiridos durante
la cursada sobre el modelo TCP/IP, el funcionamiento de los sockets TCP, la transferencia
de archivos utilizando el protocolo HTTP mediante un servidor web con la capacidad de
tanto subir (modo upload) como descargar (modo download) imagenes, videos y archivos
entre dispositivos conectados en una misma red.
En este contexto la capa de aplicación es la encargada de proporcionar tanto las interfaces
como los protocolos necesarios para establecer una comunicación entre el programa y los
clientes.
La finalidad de este servidor es escuchar las conexiones entrantes y atender las solicitudes
HTTP de clientes que iniciaron esta conexión y permanecen conectados a la misma red
para gestionar la transferencia de archivos entre el dispositivo conectado y el servidor. Para
ello se genera una URL a la que se puede acceder con un código QR.

Los detalles de la implementación, se encuentran descritos en el archivo _informe.pdf_
